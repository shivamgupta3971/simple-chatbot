## 📖 About The Project

This project is a **Simple Rule-Based Chatbot** developed using Python.
It is designed to demonstrate the use of:

- **Control Flow** (if-else statements)
- **Decision-Making Logic** (pattern matching)
- **Loops** (continuous while loop)
- **Basic AI Concepts** (rule-based response system)

The chatbot can handle greetings, answer basic questions,
tell jokes, perform simple math, show time/date,
and respond to various user inputs using predefined rules.
